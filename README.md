## Як працаваць

Бягучы стан можно паглядзець на https://konstbel.github.io/landing/

Каб запусціць сайт лакальна, выканайце ў камандным радку:

```sh
$ hugo server
```

Каб перасабраць вебпак ресурсы:

```sh
$ cd themes/ananke/src/; npm run build:production
```
